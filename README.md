# AI Finance Assistant

AI Finance Assistant is a full-stack backend application built using Spring Boot that enables users to manage, track, and analyze their personal finances. The system is designed with a scalable architecture and integrates secure authentication using JWT.

## 🚀 Features

- User Authentication (JWT-based login system)
- Add, update, and delete expenses
- Track expenses by category
- Daily expense aggregation
- User-specific financial data (secure APIs)
- OCR-based bill scanning (mock implementation)
- Automatic balance calculation (Cash/Bank)

## 🧠 Architecture

The backend follows a clean layered architecture:

Controller → Service → Repository → Database  
JWT Filter → Request Interception → User Authentication  

## 🛠 Tech Stack
